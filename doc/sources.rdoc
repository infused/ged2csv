= Sources

The record fields are defined below in the following format. 
Record Name ; Record Length ; Description

SrcKey ; 20 ; A unique number for the source record. This number is the unmodified record identifier for the master source from the source gedcom file
Text ; 65,535 ; Contains basic information about the master source
NoteKey ; 20 ; Contains additional information about the master source

Delimited File Example:
"S01583","Title: 1910 Ohio Census","NS0158"
